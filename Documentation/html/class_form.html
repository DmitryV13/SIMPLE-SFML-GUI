Detailed Description

Класс для группировки элементов со смыслом сущности формы. Предназначен для хранения элементов формы (на данный момент) на первом уровне вложенности, и обработки значений всех этих элементов при активаци формы. На уровнях вложенности больше 1 хранить элементы форм можно, но их обработка не будет осуществляться. По умолчанию форма является одноразовой - отправка произойдет только 1 раз, при первом срабатывании события тригера.
